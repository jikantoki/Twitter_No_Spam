# Twitter No Spam

![icon](icon128.png)

Twitter(X)のスパムコメントを排除、厳密には認証済みかつアカウント名に日本語が含まれていないものを非表示にします。  
また、URL や絵文字のみのリプライも非表示になるほか、単語で自動的に非表示にする機能もあります
